Pomodoro Clock

We all need to be productive

A small and simple Pomodoro clock built with JavaScript and React.JS library.

Built for fun, but you will have fun interacting with it due to its very simple UI.

Simple Pomodoro clock which fulfills the following user stories:

  • I can start a 25 minute pomodoro, and the timer will
  • go off once 25 minutes has elapsed.
  • I can reset the clock for my next pomodoro.
  • I can customize the length of each pomodoro.
